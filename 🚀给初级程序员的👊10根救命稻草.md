## qrcodejs(二维码生成器)

#### 网址
https://github.com/davidshimjs/qrcodejs

``` javascript
var qrcode = new QRCode(document.getElementById("qrcode"), {
	text: "http://juejin.im",
	width: 128,
	height: 128,
	colorDark : "#000000",
	colorLight : "#ffffff",
	correctLevel : QRCode.CorrectLevel.H
});
```
#### 兼容性
IE6~10, Chrome, Firefox, Safari, Opera, Mobile Safari, Android, Windows Mobile.
